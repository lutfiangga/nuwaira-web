# PRD - User Stories (MVP)

## Learner
- Sebagai learner, saya ingin melihat katalog course agar bisa memilih pembelajaran relevan.
- Sebagai learner, saya ingin melihat detail course dan syllabus agar bisa menilai materi sebelum enroll.
- Sebagai learner, saya ingin melanjutkan lesson terakhir agar progres belajar tidak hilang.
- Sebagai learner, saya ingin mengerjakan quiz agar tahu pemahaman saya.
- Sebagai learner, saya ingin melihat progres completion agar termotivasi menyelesaikan course.

## Instructor
- Sebagai instructor, saya ingin membuat course dan module agar bisa menyusun kurikulum.
- Sebagai instructor, saya ingin menambah lesson (teks/video/file) agar materi dapat diakses learner.
- Sebagai instructor, saya ingin membuat quiz dan passing score agar evaluasi terstruktur.

## Admin
- Sebagai admin, saya ingin mengelola role pengguna agar akses sistem aman.
- Sebagai admin, saya ingin publish/unpublish course agar kualitas konten terkontrol.
- Sebagai admin, saya ingin melihat dashboard metrik agar bisa mengambil keputusan operasional.
